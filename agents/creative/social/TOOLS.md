# Jade (Social Media Manager) — Tools & Integrations

## Core Backend Tools

### Message Bus
- Receive content assignments from Nova
- Send completed posts for review
- Flag engagement issues or community concerns

### Task Queue
- Receive social media tasks from the creative task queue
- Update task status as work progresses

### Memory Store
- Track social content output and engagement metrics
- Store platform-specific performance baselines per client

### Logger
- All social media actions logged for audit trail

### Brand Vault
- Load client brand guidelines for voice and tone
- Reference platform-specific voice adaptations

### Content Calendar
- Add and manage scheduled posts
- Track content pipeline (draft, review, approved, published)
- Maintain posting cadence per client per platform

## Social Media Capabilities

### Content Creation
- Platform-native post writing (LinkedIn, Instagram, X/Twitter, TikTok, Facebook)
- Format adaptation — carousel, single image, text, video, story, reel
- Caption writing optimised for each platform's algorithm and audience

### Content Calendar
- Calendar creation with themes, cadence, and platform distribution
- Scheduling and pipeline management
- Gap analysis — ensure consistent posting rhythm

### Engagement & Community
- Community response writing (on-brand, contextual)
- Engagement analysis — comments, shares, saves, click-throughs
- Audience sentiment monitoring

### Trends & Strategy
- Trend monitoring across platforms
- Trend-to-brand fit assessment
- Hashtag research and strategy
- Competitor social activity monitoring

## Future Integrations

- Social platform APIs (Instagram, LinkedIn, X, TikTok, Facebook)
- Social scheduling tools (Buffer, Hootsuite, Sprout Social)
- Social listening and sentiment analysis tools
- Influencer database and outreach platforms
