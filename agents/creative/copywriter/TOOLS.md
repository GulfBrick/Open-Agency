# Ash (Copywriter) — Tools & Integrations

## Core Backend Tools

### Message Bus
- Receive copy briefs and assignments from Nova
- Send completed copy for review
- Flag brand voice or compliance concerns

### Task Queue
- Receive copy tasks from the creative task queue
- Update task status as work progresses

### Memory Store
- Track copy output and performance data
- Store copy style preferences and winning variants per client

### Logger
- All copy actions logged for audit trail

### Brand Vault
- Load client brand guidelines for voice and tone
- Reference approved terminology and messaging frameworks

## Copy Capabilities

### Headline Writing
- Multiple headline variants with strategic reasoning
- A/B testing recommendations
- Platform-specific headline optimisation

### Long-Form Copy
- Landing page copy (hero, features, benefits, testimonials, CTA)
- Blog posts and thought leadership articles
- Case study and white paper copy

### Short-Form Copy
- Social media captions per platform
- Ad copy for Google, Meta, LinkedIn, TikTok
- Push notification and SMS copy

### Email
- Email sequence design and copy
- Nurture flow development
- Subject line variants and preview text
- Transactional and lifecycle email copy

### CTA & Conversion
- Call-to-action variant creation
- Button copy optimisation
- Conversion-focused microcopy

### Review & Editing
- Copy review against brief objectives and brand guidelines
- Proofreading and copy editing
- Tone and voice consistency checking

## Future Integrations

- Grammarly/LanguageTool API for automated proofing
- A/B testing platforms for copy variant performance
- SEO keyword integration tools
- Translation and localisation services
