# Iris (Graphic Designer) — Tools & Integrations

## Core Backend Tools

### Message Bus
- Receive creative briefs and assignments from Nova
- Send completed designs for review
- Flag brand consistency issues

### Task Queue
- Receive design tasks from the creative task queue
- Update task status as work progresses

### Memory Store
- Track design output (concepts, assets, specifications)
- Store design decisions and rationale per client

### Logger
- All design actions logged for audit trail

### Brand Vault
- Load client brand guidelines before every design task
- Reference colour palettes, typography, and visual rules

## Design Capabilities

### Concept Development
- Visual concept description and mood specification
- Style tile creation (colour, type, imagery direction)
- Layout composition and hierarchy planning

### Asset Specification
- Exact dimensions for all target formats
- Colour specification with hex/RGB values
- Typography specification (family, weight, size, leading)
- Spacing and grid system definition

### Brand Consistency
- Review assets against brand guidelines
- Flag deviations and suggest corrections
- Maintain visual consistency across campaigns

### Colour & Typography
- Colour palette generation with harmony rules
- Accessibility-checked contrast ratios
- Typography pairing and hierarchy recommendations

## Future Integrations

- Figma API for design file management
- Canva API for template-based production
- Adobe Creative Cloud for asset creation
- Automated brand compliance checking tools
