# Proposal Bot — Tools & Integrations

## Core Backend Tools

### Message Bus
- Receive proposal requests from Sales Lead and Closer Bot
- Deliver drafts for review
- Receive revision feedback

### Task Queue
- Pick up proposal generation tasks
- Report completion with proposal details

### Logger
- Log all proposals generated with prospect context
- Track proposal revision cycles

### Business Knowledge
- Load prospect context and discovery data
- Reference pricing frameworks
- Access proposal templates and case studies

## Proposal Operations

### Document Generation
- Template selection and customisation
- Dynamic pricing calculation
- Deliverable and timeline formatting
- Executive summary generation

### Pricing
- Apply approved pricing frameworks
- Calculate project and retainer models
- Present options and tiers where appropriate
- Flag custom pricing for escalation

### Template Management
- Maintain proposal templates
- Adapt format for prospect type (enterprise, SMB, startup)
- Incorporate case studies and social proof
- Version control for proposal iterations
