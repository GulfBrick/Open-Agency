## SPRINT WORKFLOW (src/core/sprint.js)

Create src/core/sprint.js — Sprint Manager:
- Sprint: { id, clientId, startDate, endDate, status, backlog, inProgress, done, blockers }
- Methods:
  - createSprint(clientId, durationDays) — initialises sprint
  - addToBacklog(sprintId, task) — adds task
  - startTask(sprintId, taskId, agentId) — moves to inProgress
  - completeTask(sprintId, taskId, output) — moves to done
  - addBlocker(sprintId, taskId, blocker) — flags blocker
  - getSprintReport(sprintId) — full sprint status
  - Persists to data/sprints/[clientId]/[sprintId].json
