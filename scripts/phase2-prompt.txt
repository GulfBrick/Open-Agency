### 2. Backend Agent Classes

Create src/agents/cfo.js — CFO Agent class:
- Extends a base Agent class (create src/agents/base-agent.js first)
- Uses claude-sonnet-4-5 model (worker-level, not Opus)
- System prompt: Marcus persona, CFO responsibilities
- Methods:
  - analyzeFinancials(clientId, data) — reviews financial data, returns structured report
  - generateInvoice(clientId, details) — creates invoice record in data/clients/[clientId]/invoices/
  - trackExpense(clientId, expense) — logs expense to data/clients/[clientId]/expenses.jsonl
  - forecastCashFlow(clientId, months) — simple projection based on historical data
  - generateReport(clientId, period) — weekly/monthly financial summary
  - flagAnomaly(clientId, transaction) — detects unusual spending, escalates via message bus
- Records every task via experience tracker
- Injects client context before every task

Create src/agents/cto.js — CTO Agent class:
- Uses claude-sonnet-4-5 model
- System prompt: Zara persona, CTO responsibilities
- Methods:
  - reviewArchitecture(clientId, description) — reviews tech decisions, flags concerns
  - auditSecurity(clientId, details) — security posture assessment
  - monitorSystem(clientId, metrics) — evaluates system health metrics
  - recommendStack(clientId, requirements) — suggests tech stack for a given need
  - createADR(clientId, decision) — writes Architecture Decision Record to data/clients/[clientId]/adrs/
  - estimateInfrastructureCost(clientId, requirements) — cloud cost estimation
- Records every task via experience tracker

Create src/agents/cmo.js — CMO Agent class:
- Uses claude-sonnet-4-5 model
- System prompt: Priya persona, CMO responsibilities
- Methods:
  - createCampaignBrief(clientId, goal) — generates campaign brief with KPIs
  - analyzeMktMetrics(clientId, data) — interprets marketing data, recommends actions
  - generateContentCalendar(clientId, period) — creates content plan
  - writeAdCopy(clientId, brief) — generates ad copy variants
  - seoAudit(clientId, url) — basic SEO recommendations
  - competitorAnalysis(clientId, competitors) — market positioning analysis
- Records every task via experience tracker

### 3. Base Agent Class

Create src/agents/base-agent.js:
- BaseAgent class that CFO, CTO, CMO all extend
- Constructor: (agentId, name, role, model)
- Properties: agentId, name, role, model, anthropic client
- Methods:
  - chat(messages, systemPrompt) — calls Anthropic API, returns response text
  - getContext(clientId) — calls agentProfile.getAgentContext(this.agentId, clientId)
  - recordTask(clientId, taskType, outcome, details) — calls experience.recordTask()
  - escalate(to, subject, details) — sends ESCALATION message via message bus
  - log(action, details) — calls logger
- Loads ANTHROPIC_API_KEY from env

### 4. Agent Registry

Create src/core/agent-registry.js:
- Central registry of all active agents
- Methods:
  - register(agentId, agentInstance) — registers an agent
  - get(agentId) — returns agent instance
  - list() — returns all registered agents
  - getByDepartment(department) — returns agents in a department
  - dispatch(agentId, method, ...args) — calls a method on an agent, logs it
